Título: Gestión de Fútbol (DataFlex)

Funcionalidad: Navegación por entidades (jugadores, equipos, entrenadores).

Base de Datos: MySQL.

Relaciones entre las diferentes tablas (uno a muchos), un equipo tiene muchos jugadores, un entrenador dirige a un equipo, una liga tiene muchos equipos, tambien incluye tablas intermedias para relaciones entre tablas padres e hijos, por ejemplo tabla jugador-posiciones para relacionar la tabla jugadores con la tabla posiciones.
